DVI

О Smart Calculator v 1.0

Калькулятор разработан на языке Си стандарта C11 с использованием компилятора gcc и дополнительных библиотек и модулей QT.

Установка

Сборка программы осуществляется с помощью Makefile.
Достаточно перейти в корневую папку и ввести в терминале команду:

make install

И в папке ~Desktop/build/ появится скомпилированная программа.

Удаление

Введите в терминале команду:

make uninstall

Архивация основных файлов

Введите в терминале команду:

make dist

Функции и возможности

  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации
  • Вычисление произвольных скобочных арифметических выражений в инфиксной нотации с подстановкой значения переменной x в виде числа
  • Построение графика функции, заданной с помощью выражения в инфиксной нотации с переменной x

Вычисление следующих функций:

  • cos Вычисляет косинус
  • sin Вычисляет синус
  • tan Вычисляет тангенс
  • acos Вычисляет арккосинус
  • asin Вычисляет арксинус
  • atan Вычисляет арктангенс
  • sqrt Вычисляет квадратный корень
  • ln Вычисляет натуральный логарифм
  • log Вычисляет десятичный логарифм

Выполняет следующие арифмитические операции:

  • () Скобки
  • + Сложение
  • - Вычитание
  • * Умножение
  • / Деление
  • ^ Возведение в степень
  • % Остаток от деления
  • + Унарный плюс
  • - Унарный минус

График функции

Поля ввода данных:
  • Границы по X - области отрисовки функции по оси X
  • Границы по Y - области отрисовки функции по оси Y
  • График отрисовывается по переходу в эту вкладку
